# DESIGN_SYSTEM (학교폭력해설집 법률 랜딩페이지)

## 1. 개요
전문적이고 신뢰감을 주는 변호사의 브랜드 이미지를 확립하기 위해 기획된 디자인 시스템.

## 2. Colors (컬러 팔레트)
- **Primary**: `#1a1a1a` (중후한 블랙 - 신뢰와 전문성)
- **Background Light**: `#ffffff` (기본 배경)
- **Background Dark**: `#0a0a0a` (다크 모드 메인 배경)

## 3. Typography (타이포그래피)
- **Display Font**: `Inter` (숫자, 영문 제목 체계용) + `Noto Sans KR`
- **Body Font**: `Noto Sans KR`
- 굵기 변형(Black, Bold)을 활용해 묵직하고 강렬한 Hero Title 구현.

## 4. UI Elements (UI 요소)
- **Mono Tags**: `Legal Expert` 등의 캡션에 고정폭 글꼴(Monospace) 사용하여 분석적이고 객관적인 이성 표현.
- **Glassmorphism Nav**: GNB 영역에 반투명 백드롭 필터 적용하여 현대적인 감각 추가.
- **Marquee Text**: 섹션 사이 흐르는 텍스트 애니메이션 라인을 대입하여 긴장감과 전문가의 아우라 부각.

## 5. Theme
- **Dark Mode 최적화**: 화면 상단의 달 모드 토글을 통해 부드럽게 스위칭되도록 `transition-colors duration-300` 지원.
